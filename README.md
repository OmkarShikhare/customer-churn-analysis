# Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn using SQL, Python, Machine Learning, and Power BI.

The goal is to identify:

* Why customers churn
* Which users are at high risk
* What actions can reduce churn

---

##  Key Insights

*  Overall churn rate: **6%**
*  High-risk users: **15%**
*  Churn increases significantly after **10 days of inactivity**
*  Free users have the highest churn rate

---

## 🛠 Tools Used

* **Python** (Pandas, NumPy, Scikit-learn)
* **SQL** (Data Analysis)
* **Power BI** (Dashboard & Visualization)

---

##  Project Structure

* `data/` → Raw and processed datasets
* `sql/` → SQL queries for analysis
* `python/` → Data generation + ML model
* `powerbi/` → Dashboard file
* `images/` → Dashboard screenshots

---

##  Business Impact

Based on analysis, the following actions are recommended:

*  Send notifications to inactive users (>10 days)
*  Offer discounts to high-risk users
*  Educate users about features to increase engagement
